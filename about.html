AI-Powered Coastal Risk Monitoring
Monitoring India's
7,500 km coastline
in real time
OceanGuard is a real-time coastal weather monitoring system that collects live environmental data, classifies risk conditions, sends automated alerts, and generates AI-powered intelligence briefings — all powered by open APIs and workflow automation.
How it works
1
Every hour — n8n triggers the monitoring workflow automatically
2
Open-Meteo API — live wind speed, temperature, and rainfall fetched for all 9 cities simultaneously
3
Risk classification — each city scored against calibrated thresholds: NORMAL → WATCH → WARNING → EMERGENCY
4
Cooldown guard — suppresses repeat alerts within 3 hours unless severity escalates, preventing alert fatigue
5
Google Sheets — every reading logged as a timestamped row, building a historical database
6
Telegram alerts — tiered notifications sent to subscribers when thresholds are crossed
7
AI briefing — Groq / Llama 3 generates a daily intelligence report from 7 days of trend data
8
Predictive alerts — Open-Meteo 48h forecast scanned for upcoming threshold crossings

Below 1× threshold

≥ 1× threshold

Dangerous conditions. Small vessels should return to port immediately.

Engineering features
Reliability features
→
Alert cooldown — per-city 3-hour suppression prevents notification fatigue using n8n static data
→
Tier escalation override — cooldown bypassed automatically when risk escalates (WATCH → WARNING)
→
Data validation — timestamp strings filtered from city names, preventing malformed rows from corrupting the dashboard
→
Modular workflows — monitoring and briefing run as independent n8n workflows, so one failure doesn't affect the other
Intelligence features
→
Predictive alerts — Open-Meteo 48h forecast scanned hourly to warn before threshold crossings
→
AI briefings — Llama 3 analyses 7-day patterns and generates plain-language maritime intelligence daily
→
Historical analytics — rolling 7-day averages, peak tracking, and trend direction per city
→
Coastal segmentation — Arabian Sea and Bay of Bengal tracked separately with different seasonal risk profiles
